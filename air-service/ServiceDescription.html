Air Service API Description

Methods

  • GetAirCarriers

    Inputs
    DatatypeNameDescription
    stringoriginCityNameName of the city in which the flight is originating, case sensitive, first letter capitalized, subsequent letters lowercase
    stringoriginStateTwo letter abbreviation of the State where the flight is originating, case sensitive, uppercase
    stringdestinationCityNameName of the city in which the flight is landing, case sensitive, first letter capitalized, subsequent letters lowercase
    stringdestinationStateTwo letter abbreviation of the State where the flight is originating, case sensitive, uppercase
    Description

    This method takes string values for origin and destination city and state and returns a dataset of carriers available along that route. The returned dataset contains two columns; carrierID as INT and carrierName as STRING

  • GetFlights

    Inputs
    DatatypeNameDescription
    intcarrierIDInteger ID of the carrier for which flights are being requested
    stringoriginCityNameName of the city in which the flight is originating, case sensitive, first letter capitalized, subsequent letters lowercase
    stringoriginStateTwo letter abbreviation of the State where the flight is originating, case sensitive, uppercase
    stringdestinationCityNameName of the city in which the flight is landing, case sensitive, first letter capitalized, subsequent letters lowercase
    stringdestinationStateTwo letter abbreviation of the State where the flight is originating, case sensitive, uppercase
    Description

    This method takes an integer value for the carrier ID, string values for origin and destination city and state and returns a dataset of flights available along that route. The returned dataset contains eight columns; flightCode as STRING, carrierName as STRING, OriginCity as STRING, departureTime as DateTime, flightTime as FLOAT, economyPrice as double and firstClassPrice as double
